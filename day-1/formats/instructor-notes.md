# Instructor notes — Formáty

## Timing
- Krátký blok. Držet prakticky: ukázat reálný `SKILL.md`, reálný SP JSON formatting v tenantu, reálný YAML front-matter z repa.

## Demo tipy
- SP column formatting: otevřít knihovnu → Format current view → vložit mini JSON příklad → Preview.
- Ukázat, že stejná data (title/type/den) jde napsat v JSON i YAML — rozdíl v zápisu.

## Tripwires
- YAML odsazení (mezery vs. taby) — nejčastější chyba studentů.
- JSON bez komentářů — studenti se ptají „proč mi to hlásí chybu".
